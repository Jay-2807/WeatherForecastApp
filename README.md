# Weather Forecast App 

Built Android Application using Jetpack Compose that shows a 3-day weather forecast for a city, with offline support using Room database.

---

## Features
- Search weather by city name
- Display 3-day weather forecast
- Weather icons based on conditions
- Offline access using Room database

---

##
- Kotlin
- Jetpack Compose
- MVVM Architecture
- Retrofit
- Room Database
- Weather API

---

##Online & Offline Support
- When online, data is fetched from the weather API and cached locally.
- When offline, the app displays cached weather data from Room.
- If no offline data is available, a user-friendly message is shown.
